# To-Do Application

This is a simple to-do application built with Spring Boot.

## Features

- Create, read, update, and delete to-do items.
- Paginated list of to-do items.

## API Endpoints

- `POST /todos`: Creates a new to-do item.
- `GET /todos`: Retrieves a paginated list of to-do items.
- `PATCH /todos/{id}`: Updates a to-do item by its ID.
- `DELETE /todos/{id}`: Deletes a to-do item by its ID.

## Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Lombok
- MapStruct
- Maven
- Docker
